TUGAS:

1. Di artifacts/api-server/src/routes/users.ts, cek dulu semua endpoint yang sudah ada.
   Tambahkan hanya yang BELUM ada:

   GET /users (requireAdmin):
   - Return semua user: [{ username, role, daily_target }]
   - JANGAN return password_hash

   POST /users (requireAdmin):
   - Body: { username: string, password: string, role: "contributor" | "admin" }
   - Validasi: username dan password tidak boleh kosong, role harus valid
   - Cek duplikat: jika username sudah ada → return 409: { error: "Username sudah digunakan." }
   - Hash password dengan bcrypt.hash(password, 10)
   - Insert ke scraper_users
   - Return 201: { username, role, daily_target }

   PATCH /users/:username (requireAdmin):
   - Body: { role?: "contributor" | "admin" }
   - Jika user tidak ditemukan → return 404
   - Update kolom role
   - Return: { username, role, daily_target }

   POST /users/:username/reset-password (requireAdmin):
   - Generate password baru acak: 12 karakter, kombinasi huruf + angka
     gunakan crypto.randomBytes atau Math.random dengan charset
   - Hash dengan bcrypt.hash(newPassword, 10)
   - Update password_hash di scraper_users
   - Return: { username, new_password: newPassword }
     (plain text dikembalikan SEKALI ke admin — tidak disimpan)
   - Jika user tidak ditemukan → return 404

   DELETE /users/:username (requireAdmin):
   - Jika user tidak ditemukan → return 404
   - Cegah admin menghapus dirinya sendiri:
     jika username === req.session.user.username → return 400: { error: "Tidak bisa menghapus akun sendiri." }
   - Hapus dari scraper_users
   - Return 204

2. Di artifacts/aina-scraper/src/pages/admin/users.tsx, cek yang sudah ada.
   Pastikan atau tambahkan semua fitur berikut:

   Daftar user:
   - Fetch GET /api/users dengan fetch + credentials: 'include'
   - Tabel kolom: Username, Role (badge: ungu untuk admin, abu untuk contributor), Target Harian, Aksi
   - Loading skeleton, empty state jika tidak ada user

   Tambah user baru:
   - Tombol "+ Tambah Anggota" di atas tabel
   - Saat diklik: buka Dialog (gunakan komponen Dialog dari shadcn/ui yang sudah ada)
   - Form di dalam dialog: Username (text), Password Awal (text, tampilkan — bukan type password),
     Role (Select: contributor/admin)

   - Error 409: tampilkan pesan "Username sudah digunakan." di dalam dialog
   - Error lain: toast.error(...)

   Edit role:
   - Tombol "Edit" di setiap baris
   - Buka Dialog edit dengan username (read-only) dan Select role
   - Submit: PATCH /api/users/{username}
   - Sukses: toast.success("Peran berhasil diperbarui"), tutup dialog, refresh daftar

   Reset password:
   - Tombol "Reset PW" di setiap baris (di samping Edit)
   - Konfirmasi dengan AlertDialog: "Reset password {username}?"
   - Submit: POST /api/users/{username}/reset-password
   - Sukses: tampilkan Dialog khusus dengan teks:
     "Password baru untuk {username}:"
     [kotak dengan font mono, teks password, tombol salin]
