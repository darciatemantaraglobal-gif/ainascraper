# Autentikasi

Amankan akses ke aplikasi hanya untuk anggota tim AINA.

## Spesifikasi

### Tujuan
Menjaga keamanan aplikasi dengan memastikan hanya anggota tim AINA yang memiliki akun dapat masuk, mengelola sesi, dan memperbarui kredensial mereka.

### Selesai bila
- Seluruh halaman aplikasi (kecuali login) tidak bisa diakses tanpa sesi aktif; pengguna yang belum login diarahkan ke halaman login.
- Pengguna dapat membuktikan identitasnya melalui form username dan password, lalu sistem mengarahkan ke dashboard sesuai peran.
- Tombol "Keluar" tersedia di antarmuka untuk mengakhiri sesi kapan saja.
- Pengguna dapat mengubah password mereka sendiri melalui halaman khusus setelah login.

## Sub-fitur: Login

Masuk ke aplikasi dengan username dan password.

### Tujuan
Memungkinkan anggota tim yang telah didaftarkan oleh admin untuk masuk ke aplikasi menggunakan username dan password yang diberikan.

### Selesai bila
- Form login menampilkan kolom input "Username" dan "Password" serta tombol "Masuk".
- Setelah menekan "Masuk" dengan data valid, pengguna langsung diarahkan ke halaman "Mulai Input" untuk kontributor atau "Dashboard Admin" untuk admin, tanpa langkah tambahan.
- Jika username atau password salah, muncul pesan "Username atau password salah. Silakan coba lagi." di bawah form.
- Saat sesi login valid, mengunjungi halaman login otomatis mengalihkan ke dashboard yang sesuai.

## Sub-fitur: Logout

Keluar dari sesi yang sedang berjalan.

### Tujuan
Mengakhiri sesi pengguna yang sedang berjalan sehingga aplikasi tidak dapat diakses tanpa login ulang.

### Selesai bila
- Tombol "Keluar" tersedia di pojok kanan atas setiap halaman setelah login.
- Mengklik "Keluar" langsung menghapus sesi dan mengarahkan pengguna ke halaman login.
- Setelah logout, halaman dashboard tidak dapat diakses kembali sebelum login ulang (mencoba akses langsung menampilkan halaman login).

## Sub-fitur: Ganti Password

Ubah password akun sendiri.

### Tujuan
Memberi pengguna kemampuan mengubah password sendiri demi keamanan akun.

### Selesai bila
- Halaman "Ganti Password" dapat diakses dari menu pengguna (misal, ikon profil di pojok kanan atas).
- Form berisi tiga kolom: "Password Lama", "Password Baru", "Konfirmasi Password Baru", dan tombol "Simpan Perubahan".
- Mengisi password lama yang benar, password baru dan konfirmasi yang cocok, lalu menyimpan, akan memperbarui password dan menampilkan notifikasi sukses "Password berhasil diubah."
- Jika password lama salah, muncul pesan "Password lama tidak sesuai." tanpa mengubah password.
- Jika konfirmasi password baru tidak cocok, muncul pesan "Konfirmasi password tidak sama."
