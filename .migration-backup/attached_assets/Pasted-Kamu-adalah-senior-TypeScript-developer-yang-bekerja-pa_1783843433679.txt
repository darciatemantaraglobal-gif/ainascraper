2. Buat file baru artifacts/api-server/src/lib/contentProcessor.ts berisi:

   function cleanText(text: string): string
   - Hapus semua tag HTML: ganti <[^>]+> dengan ""
   - Hapus karakter non-standar: hanya sisakan huruf, angka, tanda baca umum, spasi, newline
   - Rapikan whitespace: ganti 3+ newline berturut → 2 newline, spasi ganda → spasi tunggal
   - Trim hasil akhir
   - Export fungsi ini

   function toMarkdown(title: string, content: string): string
   - Baris pertama: # {title}
   - Pisah content by paragraf (split \n\n atau \n)
   - Paragraf yang diawali angka+titik (1. 2. dst) → biarkan sebagai numbered list
   - Paragraf yang diawali - atau * → biarkan sebagai bullet list
   - Paragraf biasa → tulis apa adanya, pisahkan dengan baris kosong
   - Return string Markdown lengkap
   - Export fungsi ini

   function chunkText(text: string, chunkSize = 1000, overlap = 100): string[]
   - Split teks menjadi array chunk dengan ukuran maksimal chunkSize karakter
   - Usahakan potong di batas kalimat (cari ". " atau "\n" terdekat sebelum chunkSize)
   - Overlap: setiap chunk baru dimulai overlap karakter sebelum akhir chunk sebelumnya
   - Jika teks lebih pendek dari chunkSize, return [text] (satu chunk saja)
   - Export fungsi ini

   async function generateEmbedding(text: string): Promise<string | null>
   - POST ke https://openrouter.ai/api/v1/chat/completions
   - Model: "text-embedding-ada-002" via OpenRouter — gunakan endpoint embeddings jika tersedia,
     jika tidak tersedia di OpenRouter pakai model "openai/text-embedding-ada-002"
   - Endpoint embeddings OpenRouter: POST https://openrouter.ai/api/v1/embeddings
     body: { model: "openai/text-embedding-ada-002", input: text }
   - Return JSON.stringify(response.data[0].embedding) — simpan sebagai string JSON
   - Jika OPENROUTER_API_KEY tidak ada atau request gagal → return null (jangan throw)
   - Timeout: 10 detik dengan AbortController
   - Export fungsi ini
